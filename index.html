🚗Étape 1A: Status Transition → EN ROUTE

1A

EN ROUTE

Avant de quitter le bureau

  • Ouvre ton appel de service
  • Clique sur Actions
  • Sélectionne Status Transition
  • Choisis En route
  • Clique Suivant
✓ Résultat
  • SMS automatique envoyé au client
  • Lien GPS temps réel partagé
  • Client peut te suivre
  • Heure estimée affichée
1B

EN COURS

À l'arrivée chez le client

  • Arrive chez le client
  • Clique Actions
  • Sélectionne Status Transition
  • Choisis En cours
  • Clique Suivant
✓ Résultat
  • Heure d'arrivée enregistrée
  • GPS désactivé pour le client
  • Message affiché au client
  • Travail commence

⚠️ IMPORTANTE

NOTE MANUELLEMENT L'HEURE D'ARRIVÉE SUR PAPIER! L'app ne peut pas la modifier après. 15-30 min oubliées = perte énorme pour l'entreprise.

📸Étape 2: Capture d'image

Quoi photographier

Dès ton arrivée sur place

  • ✓ État avant réparation
  • ✓ Le problème identifié
  • ✓ Les réparations effectuées
  • ✓ L'emplacement/contexte
  • ✓ Vue d'ensemble + gros plans

Étapes dans l'app

Procédure

  • 1. Clique Actions
  • 2. Capture d'image
  • 3. Prends photos
  • 4. OPTIONNEL: Edit Image pour annoter
  • 5. Clique Add to Record
✓ Photos sauvegardées
  • Dans ASSOCIÉS > Fichiers
  • Accessibles au client

⚠️ CRITIQUE: UNE FOIS 'ADD TO RECORD' = IRRÉVERSIBLE

Les photos ne peuvent plus être modifiées, supprimées ou changées après. Vérifies-les avant de confirmer!

🔧Étape 3: Consommation de produit

EXEMPLE SCENARIO

Pompe spa brisée → tu la remplaces par une pompe neuve POMPE SUPERMAX. Prix: 2199,99$

3A: Recherche

Trouver le produit

  • Scanner le code-barre de la pompe neuve
  • OU Recherche par nom: "POMPE SPA"
  • OU Cherche dans la liste complète

3B: Sélection

Choisir le produit

  • Appuie sur le rectangle sous "Produit"
  • Liste déroulante s'ouvre
  • Sélectionne ta pompe
  • Le produit s'affiche
  • Clique Suivant

3C: Quantité, Prix, Garantie

Détails du produit

  • Quantité: 1 pompe
  • Prix unitaire: 2199,99$
  • État d'exécution: EXÉCUTÉ
  • Garantie: Non (sauf si applicable)

3D & 3E: Résumé & Retour

Finir le produit

  • App affiche montant total: 2199,99$
  • Question: "Produit retourné?" → OUI
  • Quantité retournée: 1 (l'ancienne pompe)
  • Description: "Pompe défectueuse - fuites"
  • État d'exécution: EXÉCUTÉ

💪Étape 4: Ajouter la main d'œuvre

EXEMPLE DE CALCUL

Arrivée: 12h03 | Fin: 12h45 | Temps travaillé: 42 min = 0.7h | Tarif: 180 $/h | Montant: 126 $

Calcul du montant

Basé sur l'heure notée manuellement

Élément Calcul Résultat
Temps travaillé 12h45 - 12h03 42 min
En heure 42 ÷ 60 0.7h
Tarif horaire Trévi standard 180 $/h
MONTANT FINAL 0.7 × 180 126 $

Étapes dans l'app

Enregistrement

  • 1. Clique Actions
  • 2. Ajouter la main d'œuvre
  • 3. Montant: 126 $
  • 4. Description: Laisser vide (optionnel)
  • 5. Garantie: Non
  • 6. État d'exécution: EXÉCUTÉ
  • 7. Clique Suivant
✓ Main d'œuvre enregistrée
  • 126 $ facturés au client
  • Temps enregistré

⚠️ TRÈS IMPORTANT

L'heure d'arrivée NE PEUT PAS être modifiée dans l'app! Tu DOIS la noter manuellement sur papier chez le client. 15-30 minutes oubliées = perte énorme pour l'entreprise au bout d'une année.

💬Étape 5: Ajouter vos commentaires

1️⃣ Commentaires AU CLIENT

VISIBLE dans le rapport

Exemple à rédiger:

"Bonjour Jean, nous avons identifié que le joint d'étanchéité de votre pompe était défaillant. Nous avons remplacé la pompe complète par une POMPE SUPERMAX neuve, garantie 3 ans. L'installation est complétée et testée. Votre système fonctionne normalement. Merci!"

2️⃣ Notes INTERNES (Bureau)

INVISIBLE au client

Exemple:

"Pompe remplacée sans problème. Client satisfait. Ancienne pompe rapportée pour recyclage. RAS."

Étapes dans l'app

Enregistrement

  • 1. Clique Actions
  • 2. Ajouter des commentaires
  • 3. Remplis "Commentaires au client"
  • 4. Remplis "Notes internes (bureau)"
  • 5. Clique Suivant

Cas spéciaux

Si statut différent

Si le statut est:

  • Ne peut pas complété
  • Besoin d'un autre rdv
  • Analysé par le bureau

→ Expliquer la raison dans Notes internes!

Affichage du résumé

✓ MAIN D'ŒUVRE - Quantité 1 - Prix 126$ - EXÉCUTÉ

Vérification

Avant de continuer

  • ✓ Tous les montants corrects?
  • ✓ Rien à ajouter?
  • ✓ Produits bien listés?
  • ✓ Main d'œuvre correcte?
Si tout est OK:
  • Clique Suivant

Si correction nécessaire

Modifier une ligne

  • 1. Sélectionne la ligne à modifier
  • 2. Choisis l'action:
  • → EXÉCUTÉ (facturer)
  • → BROUILLON (estimé seulement)
  • → RÉVOQUÉ (supprimer)
  • 3. Clique Suivant

🚨 DANGER: 'RÉVOQUÉ'

'RÉVOQUÉ' = supprime TOUT (produits + garanties + main d'œuvre d'un coup). Utiliser avec caution!

📄Étape 7: Générer le rapport de service

Étape 1: Validation

Vérifier la facturation

  • 1. Clique Actions
  • 2. Générer le rapport de service
  • 3. Vérifies que tout est bon
  • 4. Si erreur → reviens aux étapes précédentes

Étape 2: Signature CLIENT

Récupérer la signature

  • Section "Signature du client"
  • Clique "Ajouter une signature"
  • Client signe avec son doigt
  • Clique "Enregistrer"

Si client absent

Alternative

  • Clique "Impossible de récupérer la signature"
  • Indique la raison:
  • → Client absent
  • → Refus de signer
  • → Autre raison

Étape 3: Signature TECHNICIEN

Votre signature (toi)

  • Section "Signature du technicien"
  • Clique "Ajouter une signature"
  • SOIT: Coche "Utiliser mon nom"
  • SOIT: Signe manuellement
  • Clique "Enregistrer"
  • Clique "Générer"
✓ Rapport généré!
  • Prêt à être envoyé
  • Signatures enregistrées

💰Étape 8: Paiement

⚠️ AVANT TOUT

Rafraîchis ou synchronise l'app! Cela évite que les items soient périmés ou manquants dans le calcul.

Envoyer le lien de paiement

Procédure complète

  • 1. Clique Actions
  • 2. Sélectionne Paiement
  • 3. Vérifie montant: 2667,41$ (après taxes)
  • 4. Sélectionne contact: Jean Dupont
  • 5. Clique Suivant
✉️ Email envoyé au client
  • Avec lien de paiement sécurisé
  • Client peut payer par carte
  • Ou par virement directement

Si plusieurs contacts

Distribuer la facturation

  • Tu peux sélectionner plusieurs contacts
  • Exemple: Jean (client) + gestionnaire
  • Tous recevront l'email avec lien

🚨Erreurs à absolument éviter

❌ Mettre 'Complété' par erreur

Appel le BUREAU immédiatement. Les actions faites après ne seront pas transférées.

❌ Oublier l'heure d'arrivée

❌ Modifier photos après 'Add to Record'

C'est impossible. Vérifier avant de confirmer.

Brouillon = estimé seulement. Il faut EXÉCUTÉ pour facturer.

'RÉVOQUÉ' supprime TOUT d'un coup. Double-vérifier avant!
